Help - Online Billing System

System Overview

The Online Billing System helps Pahana Edu efficiently manage customer accounts, item details, and billing processes. This system replaces manual account maintenance with a secure, user-friendly web application.

Functionalities

  • User Authentication: Log in securely with your username and password to access the system.
  • Add New Customer Accounts: Register new customers by entering their account number, name, address, telephone number, and units consumed.
  • Edit Customer Information: Update existing customer details as necessary.
  • Manage Item Information: Add, update, or delete items available in the system.
  • Display Account Details: View detailed customer account information at any time.
  • Calculate and Print Bill: Automatically compute the bill based on units consumed and print billing statements.
  • Help Section: Access system usage guidelines (this page) for assistance.
  • Exit System: Safely log out or close the application when done.

How to Use

  1. Login: Enter your username and password on the login page and click "Login".
  2. Adding Customers: Navigate to "Add Customer", fill in the required details, and submit the form to save a new account.
  3. Editing Customers: Search for the customer by account number, update any details, and save changes.
  4. Managing Items: Access the item management section to add new items, update existing ones, or remove obsolete items.
  5. Viewing Accounts: Use the account search function to display customer details and consumption history.
  6. Billing: Select the customer and generate the bill based on units consumed. You can print or save the bill as needed.
  7. Logout/Exit: Always use the logout button to end your session securely.
